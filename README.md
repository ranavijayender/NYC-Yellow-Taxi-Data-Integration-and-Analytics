# NYC-Yellow-Taxi-Data-Integration-and-Analytics
Data Integration and Analytics project using Apache spark, PySpark, Spark SQL and Databricks.
NYC Yellow taxi trip records:
https://www.nyc.gov/site/tlc/about/tlc-trip-record-data.page

NOTE: The monthly NYC Yellow Taxi Parquet datasets used in this project are publicly available on the provided link. These could not be added to this repository due to their large file size
